Excel Auditor

Free Office.js add-in for Excel on Mac. Audit precedents, dependents, and formula chains across an entire workbook.

Install

Three ways. Pick what matches your comfort level.

Option A — Ask your AI (easiest)

Click the button, paste into ChatGPT, Claude, Gemini, or any other AI chat. It will walk you through every click.

I want to install a free Excel for Mac add-in called "Excel Auditor". It is open source and lives at https://github.com/andresjz83/excel-auditor.

I am using macOS. Please walk me through the install step by step. Be specific about which Finder menus to open and which keys to press. Don't assume I know how to use Terminal or how to create folders by hand. Use Finder where possible.

Two things need to happen on my Mac:

1. The folder at this path needs to exist: ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef/ — create it if it doesn't already exist.

2. The manifest file at https://andresjz83.github.io/excel-auditor/manifest.xml needs to be downloaded into that folder, with the filename "excel-auditor-manifest.xml".

After those two things:
- I quit Excel fully (Cmd+Q) and reopen it.
- I open the auditor each Excel session via Insert → Add-ins → Excel Auditor in Excel's menu bar.

(If you happen to be a coding agent like Claude Code or Cursor with access to my computer, feel free to just run the install yourself. Otherwise walk me through it in chat.)

Option B — Click-to-install (no terminal knowledge needed)

Download a tiny installer that creates the folder and places the manifest for you.

Download installer
  1. Double-click excel-auditor-install.zip in your Downloads.
    macOS unzips it automatically. A file called install.command appears next to it.
  2. Right-click install.command → Open.
    macOS will warn that it's from an unidentified developer. Click Open. (This warning only appears the first time.)
  3. A Terminal window pops up, runs the install, then asks you to press Enter.
    Press Enter to close. The install is done.
  4. Quit Excel (⌘Q), reopen it.

Option C — Manual install (if you prefer to see what's happening)

Download manifest.xml
  1. Open Finder. Press ⌘⇧G.
    A path bar appears. Paste this exact path and press Enter:
    ~/Library/Containers/com.microsoft.Excel/Data/Documents/
  2. Look for a folder named wef inside this folder.
    If you see it: double-click to open it.
    If you don't see it: right-click inside the Finder window (on the empty white area) → New Folder → name the folder wef (lowercase, no spaces). Then double-click into it.
  3. Drag the downloaded excel-auditor-manifest.xml into the wef folder.
  4. Quit Excel (⌘Q), reopen it.

Using it

Every Excel session, open the auditor from Insert → Add-ins → Excel Auditor. The sidebar appears on the right.

Excel for Mac doesn't pin sideloaded add-ins to the Home ribbon, so this one-click step is needed each time you launch Excel. After that, the auditor works normally for the whole session.

Updates

You never reinstall. The add-in loads its code from this site, so improvements show up automatically the next time you open the pane. The version shown in the pane header tells you what you're running.

Online-only. The add-in's code is hosted on GitHub Pages. Excel fetches it from the internet each time you open the pane, which is also why it stays up to date on its own. For an offline install that runs everything from your machine, see the local install option on GitHub. The offline version only updates when you pull the latest code.

What it does

  • Multi-cell precedent tracing
  • Multi-cell dependent tracing
  • Native trace arrows on the sheet
  • Tree view in formula order
  • Formula highlight on focus
  • ↑/↓ arrow navigation
  • Esc clears, doesn't close
  • Copy formula button
  • Safe-to-delete mode
  • Opacity warnings (INDIRECT/OFFSET)
  • Free and open source

Uninstall

Delete excel-auditor-manifest.xml from the same folder you dropped it into. Restart Excel.
