Excel Auditor

Free Office.js add-in for Excel on Mac. Audit precedents, dependents, and formula chains across an entire workbook.

Download manifest.xml

Install in 3 steps

  1. Open the Excel sideload folder in Finder.
    Press ⌘⇧G in Finder and paste this path:
    ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef/
    If the wef folder doesn't exist, create it.
  2. Drop the downloaded excel-auditor-manifest.xml into that folder.
    That's the entire install. No terminal, no Node, nothing.
  3. Quit Excel completely (⌘Q), then reopen it.
    The Home ribbon now has an Open Auditor button. Click it.
Online-only. The add-in's code is hosted on GitHub Pages. Excel fetches it from the internet each time you open the pane. For an offline-capable install that runs everything from your machine, see the local install option on GitHub.

What it does

  • Multi-cell precedent tracing
  • Multi-cell dependent tracing
  • Tree view with sheet grouping
  • Formula highlight on focus
  • ↑/↓ arrow navigation
  • Esc clears, doesn't close
  • Copy formula button
  • Safe-to-delete mode
  • Opacity warnings (INDIRECT/OFFSET)
  • Free, MIT, open source

Uninstall

Delete excel-auditor-manifest.xml from the same folder you dropped it into. Restart Excel.
